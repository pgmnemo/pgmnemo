## 3. v0.3.0 Changes Relevant to Recall

The v0.3.0 migration (`extension/pgmnemo--0.2.1--0.3.0.sql`) makes the following
changes that affect retrieval behavior:

| Section | Change | Expected Recall Impact |
|---------|--------|----------------------|
| S1–S4   | Add `edge_kind` ENUM column to `mem_edge`, backfill from `relation_type` | No direct impact on recall |
| S5–S6   | Replace `ix_pgmnemo_mem_edge_type_time` with per-kind partial indexes | Minor query plan improvement; recall neutral |
| S7      | `recall_lessons()` BFS: `edge_type IN ('causal','semantic')` → `edge_kind IN ('causal','temporal')` | **Potential recall change**: `temporal` edges now included in BFS; `semantic` edges excluded |
| S8      | `traverse_causal_chain()`: `edge_type = 'causal'` → `edge_kind = 'causal'` | Recall impact on causal-chain queries |

**Hypothesis:** Inclusion of `temporal` edges (replacing `semantic`) in `recall_lessons()`
BFS should improve temporal-category recall. The overall recall@10 target is ≥ 0.795 (non-regression).

## 4. Significance Test Plan

When the bench is executed, use a paired Wilcoxon signed-rank test on per-question
recall@10 scores (v0.3.0 vs v0.2.1, n=1982 pairs). Reject null (no difference) at
p < 0.05 (two-sided).

Additionally compute:
- Cohen's d for effect size on recall@10 and MRR deltas
- Bootstrap 95% CIs on the delta (10,000 resamples)
- Regression gate: v0.3.0 recall@10 ≥ 0.795 − 0.010 (−1 pp tolerance per PROTOCOL.md §6)
- Regression gate: v0.3.0 MRR ≥ 0.548 − 0.010

### Step 5 — Compare and Significance Test

```bash
python - <<'EOF'
import json
from scipy import stats
import numpy as np

v021 = json.load(open("benchmarks/locomo/results/v0.2.1_session_20260509/metrics.json"))
v030 = json.load(open("benchmarks/locomo/results/v0.3.0_<date>/metrics.json"))

# Load per-question recall@10 from raw_retrievals.jsonl
def load_recalls(path):
    recalls = []
    with open(path) as f:
        for line in f:
            r = json.loads(line)
            recalls.append(r.get("recall_at_10", 0.0))
    return recalls

r021 = load_recalls("benchmarks/locomo/results/v0.2.1_session_20260509/raw_retrievals.jsonl")
r030 = load_recalls("benchmarks/locomo/results/v0.3.0_<date>/raw_retrievals.jsonl")

stat, p = stats.wilcoxon(r030, r021, alternative="two-sided")
delta = np.mean(r030) - np.mean(r021)
print(f"v0.2.1 recall@10: {np.mean(r021):.4f}")
print(f"v0.3.0 recall@10: {np.mean(r030):.4f}")
print(f"Delta:            {delta:+.4f}")
print(f"Wilcoxon p-value: {p:.4f}")
print(f"Regression gate:  {'PASS' if np.mean(r030) >= 0.785 else 'FAIL'}")
EOF
```
