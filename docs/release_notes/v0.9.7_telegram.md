<b>pgmnemo v0.9.7</b> — MCP-сервер догоняет v0.9.6

MCP-only релиз: схема не изменилась, только сервер.

<b>Новые параметры в pgmnemo.recall:</b>

• <code>role_filter</code> и <code>project_id_filter</code> — скоуп по роли агента и проекту
• <code>exclude_dag_id</code> — агент не recall'ит уроки, которые сам породил в этом запуске

<b>Новые параметры в pgmnemo.ingest:</b>

• <code>item_kind</code> — тип контента: <code>note</code>, <code>skill_md</code>, <code>template</code>, <code>script</code>, <code>reference</code>, <code>config</code>, <code>spec</code>
• <code>source_dag_id</code> — id воркфлоу, который создал урок

<b>Новый инструмент pgmnemo.patch:</b>

Обновляет существующий урок на месте — инкрементирует <code>version_n</code> и <code>patch_count</code>. Для случаев, когда урок устарел и нужна правка, а не новый дубль.

<b>Обновление MCP-сервера:</b>
<pre>pip install --upgrade pgmnemo-mcp</pre>

Расширение PostgreSQL не менялось — <code>ALTER EXTENSION</code> не нужен.
